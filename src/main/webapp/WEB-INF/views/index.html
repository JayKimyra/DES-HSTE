Короткий гайд как пользоваться всем этим добром

Авторизация и регистрация справа вверху. Процесс достаточно простой. Для неавторизованных пользователей есть возможность решения вариантов и просмотра заданий. Для всех остальных есть возможность создавать эти самые варианты и задания.

К тому же можно добавляться к существующим пользователям, конкретнее становится учеником или учителем для них. Будучи учителем можешь задать дз в виде варианта. У ученика отобразиться дз, он может его сделать, в таком случае результат выполнения варианта сохранится.

Учитель может править задания, если вдруг они случайно сохранились как неправильные или в случае если это задание с рисунком, хотя по факту являются правильными.

Если вдруг лень создавать собственные задания и просто хочется протестить как всё работает, то нажимай на кнопку. Автоматически создается несколько заданий, пользователей и 1 вариант.
